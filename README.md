# Financial Services Customer Churn Analysis

##  Context
Predicting customer turnover (churn) is a critical task for financial institutions. This project analyzes behavioral patterns to identify high-risk customers before they leave.

##  Methodology
- **EDA (Exploratory Data Analysis):** Understanding customer tenure and spending habits.
- **Modeling:** Using `Gradient Boosting` to handle non-linear relationships in customer data.
- **Insights:** Providing data-driven suggestions for customer retention.

##  Tech Stack
- **Python** (Pandas, Scikit-learn)
- **Data Science Workflow:** Feature selection and model evaluation.
